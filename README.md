## What is this project?

Algots is a C++ program that sorts a collection of 100,000 English words using three different algorithmic strategies, and compares their execution time and memory usage.

The three strategies are:
- **QuickSort** — over a dynamic array (`std::vector<string>`)
- **HeapSort** — using a Binary Max-Heap
- **AVL Tree** — with inorder traversal to obtain the sorted result

---

## Project Structure

```
Algots/
├── main.cpp          ← Entry point. Orchestrates dataset creation and all three algorithms
├── quicksort.h       ← QuickSort implementation (partition + quickSort)
├── heapsort.h        ← HeapSort implementation (heapify + heapSort)
├── avl.h             ← AVL Tree implementation (insert + inorder + rotations)
├── words_alpha.txt   ← Source word list (must be placed in cmake-build-debug/)
├── dataset.txt       ← Auto-generated by the program (100,000 shuffled words)
└── README.md         ← This file
```
---

## How to run
1. Open the project in CLion
2. Place `dataset.txt` inside `cmake-build-debug/`
3. Click **Run**
4. The program will:
   - Read `dataset.txt` and create `basedatos.txt` with 100,000 shuffled words
   - Run QuickSort, HeapSort, and AVL Tree on the dataset
   - Print execution times and memory estimates for each algorithm

## Algorithm Summary

| Algorithm | Time Complexity | Space Complexity | Practical Speed |
|-----------|----------------|------------------|-----------------|
| QuickSort | O(n log n) avg | O(log n)         |  Fastest       |
| HeapSort  | O(n log n) always | O(1)          |   Medium        |
| AVL Tree  | O(n log n) always | O(n)          |  Slowest       |
